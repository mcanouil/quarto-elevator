Elevator.js Quarto Extension

This extension allows you to use Elevator.js in your Quarto HTML documents. It provides an {{< elevator >}} shortcode:

  • Mandatory:

    {{< elevator >}}

  • Optional <text-button>, <anchor-target>, and <audio=audio.mp3>:

    {{< elevator <text-button> <anchor-target> <audio=audio.mp3> >}}
